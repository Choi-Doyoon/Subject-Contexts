1.네트워킹 기초:
InetAddress 클래스는 IP 주소를 다룰 때 유용하며, getLocalHost()와 getByName() 메서드로 로컬 및 원격 주소를 얻을 수 있습니다. 예제로 구글과 네이버의 IP를 확인하는 코드가 포함되어 있습니다.

2.소켓 프로그래밍:
Socket과 ServerSocket 클래스는 클라이언트-서버 통신을 위한 기본 클래스입니다. Socket 객체는 서버와 연결을 설정하고, ServerSocket은 연결 요청을 수락합니다.
예제 코드로 EchoServer와 EchoClient가 포함되어 있어, 간단한 메시지 교환을 구현할 수 있습니다.

3.URL과 URLConnection:
URL 클래스는 웹 주소를 다루기 위해 사용하며, 프로토콜, 호스트, 포트 등을 구분하여 출력하는 예제가 포함되어 있습니다.
URLConnection 클래스는 URL 연결을 열고, 데이터 타입, 길이, 수정 시간 등의 메타 데이터를 확인할 수 있습니다.

4.HttpURLConnection 및 HttpClient:
HttpURLConnection은 HTTP 요청을 처리하는 데 쓰이며, 요청 메서드나 응답 코드를 확인하는 기능을 포함합니다.
HttpClient는 더 현대적인 HTTP 통신 방식을 지원하며, HttpRequest와 HttpResponse를 사용해 요청과 응답을 관리할 수 있습니다. 응답 코드와 헤더 정보를 읽는 예제가 포함되어 있습니다.

5.DatagramSocket과 DatagramPacket:
UDP 통신을 위한 DatagramSocket과 DatagramPacket 클래스를 통해 비연결 지향 방식의 데이터 전송이 가능합니다. 서버와 클라이언트가 메시지를 주고받는 예제가 포함되어 있습니다.
